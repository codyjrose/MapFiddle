Mapstrap was created to assist developers by providing much of the boilerplate code needed to create a web map. As well as providing working documentation for often used map features.

How to use Mapstrap
  1. Play with Options, Features and Events
  2. Zoom and center the map to the desired location
  3. Click the HTML + Javascript button to get the code to make the map.

The HTML and JavaScript displayed in the output window can be copy/pasted into a lone .html file for a fully working map.

This project is open source and can be found on GitHub
